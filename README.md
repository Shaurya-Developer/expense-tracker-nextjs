## Overview

A full-stack expense tracker to:

- Add expenses
- View expenses
- Filter by category
- Sort by date (newest/oldest)
- View total of visible expenses

---

## Tech Stack

- Next.js (Frontend + Backend)
- JavaScript
- Supabase (PostgreSQL)
- Vercel (Deployment)

---

## Features

- Create expense
- Filter by category
- Sort by date (asc/desc)
- Total calculation
- Handles multiple submissions & refresh

---

## How to Run

```bash
npm install
npm run dev
```
